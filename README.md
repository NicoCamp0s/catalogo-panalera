# Catálogo Pañalera Nico

Aplicación web de catálogo para Pañalera Nico, construida con Next.js y Tailwind CSS. Permite ver productos, buscarlos por nombre o descripción, navegar por páginas y contactar por WhatsApp o Instagram.

## Scripts

- `npm run dev` — inicia el servidor de desarrollo.
- `npm run build` — genera la build de producción.
- `npm run start` — levanta la aplicación compilada.

## Características principales

- Catálogo de productos con búsqueda inteligente.
- Paginación para navegar entre productos.
- Accesibilidad básica en el buscador y botones de navegación.
- Integración con WhatsApp e Instagram para consultas.
